# Movies - API

API simple que maneja usuarios (creacion y login), permite listar peliculas de la api de iTunes. Y poder realizar comentarios sobre alguna pelicula.

## Tecnologias utilizadas:

- Nodejs (Servidor)
  - axios
  - bcrypt
  - dotenv
- Go
  - crypto
  - godotenv
  - mux
  - pq
- PostgreSQL (BD)
- JWT (Autenticacion)
- Swagger (Documentacion)
